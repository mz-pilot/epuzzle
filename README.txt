EPUZZLE - LOGIC PUZZLE SOLVER
=============================

A versatile command-line application designed to solve complex logic puzzles 
(e.g., Einstein's/Zebra Puzzle, Sudoku variants, etc.).

QUICK START
-----------

1. EXAMPLES:
   A directory named "puzzle_examples" is included with this distribution.
   It contains several ready-to-use input files to quickly test the solver.

2. USAGE INSTRUCTIONS:
   To view all available options, usage syntax, and quick examples, run the 
   application directly in your terminal:
   
   epuzzle --help
